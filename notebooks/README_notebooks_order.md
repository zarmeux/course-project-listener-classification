# Порядок запуска ноутбуков

Новая версия проекта предполагает последовательный запуск пяти ноутбуков:
1. `01_clean_and_time_features.ipynb` — очистка исходных данных и построение временных признаков;
2. `02_activity_datasets.ipynb` — построение таблиц активности;
3. `03_user_features.ipynb` — построение таблицы `user_features.csv`;
4. `04_listener_types.ipynb` — присвоение типов слушателей и временных профилей;
5. `05_report_and_checks.ipynb` — построение итоговых графиков, summary-таблиц и финальных проверок.
