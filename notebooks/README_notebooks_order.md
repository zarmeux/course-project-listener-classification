# Порядок запуска ноутбуков

Рекомендуемый порядок запуска ноутбуков:
1. `01_clean_and_time_features.ipynb` — очистка исходных данных и построение временных признаков;
2. `02_activity_datasets.ipynb` — построение таблиц активности;
3. `03_user_features.ipynb` — построение таблицы `user_features.csv`;
4. `04_listener_types.ipynb` — присвоение типов слушателей и временных профилей;
5. `05_report_and_checks.ipynb` — построение итоговых графиков, сводных таблиц и финальных проверок.
6. `06_types_sensitivity_check.ipynb` — отдельная проверка устойчивости классификации на сокращённом временном окне `2021-01-01` – `2021-01-25`.

Ноутбук `06_types_sensitivity_check.ipynb` запускается после основного пайплайна и не перезаписывает базовые итоговые таблицы в `data/processed/`.
