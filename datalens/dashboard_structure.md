# Структура дашборда Yandex DataLens

Интерактивный дашборд построен в Yandex DataLens на основе подготовленных CSV-файлов из папки `data/processed/`.

Дашборд состоит из трёх страниц.

## Страница 1. Общий обзор

Страница предназначена для общего обзора исследуемой выборки и первичного анализа динамики пользовательской активности.

Основные элементы страницы:

- общее число прослушиваний, пользователей, активных дней и сессий;
- динамика числа событий прослушивания по календарным датам;
- распределение прослушиваний по часам суток;
- распределение прослушиваний по будням, выходным и дням недели;
- краткая сводка по общей активности пользователей.

Используемые данные:

- таблица отдельных прослушиваний;
- таблица сессий прослушивания;
- таблица пользовательской активности.

## Страница 2. Временные паттерны

Страница предназначена для анализа временной структуры прослушиваний внутри суток, недели и календарного месяца.

Основные элементы страницы:

- распределение прослушиваний по частям суток;
- распределение прослушиваний по часам суток;
- распределение прослушиваний по дням недели;
- распределение прослушиваний по будням и выходным;
- тепловая карта (сводная таблица) активности в разрезе дня недели и часа суток;
- дневная активность пользователей по календарным датам.

Используемые данные:

- таблица отдельных прослушиваний;
- таблица дневной активности пользователей;
- таблица активности по сочетанию дня недели и часа.

## Страница 3. Типы слушателей

Страница предназначена для представления итоговой классификации пользователей по типам слушательского поведения и временным профилям активности.

Основные элементы страницы:

- распределение пользователей по основным типам слушателей;
- распределение пользователей по поведенческим профилям;
- сопоставление регулярности и интенсивности прослушивания;
- сопоставление ночной и выходной активности пользователей;
- итоговая таблица пользовательских характеристик.

Используемые данные:

- итоговая таблица пользовательских признаков и типов слушателей.
