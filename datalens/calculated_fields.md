# Вычисляемые поля DataLens

## Назначение документа

Документ описывает вычисляемые поля, используемые при построении дашборда в Yandex DataLens.

## Основные показатели

## Количество прослушиваний

Используется в числовых индикаторах и графиках на основе таблицы `listenings_datalens.csv`.

```text
COUNT([listening_id])
```

## Количество пользователей

Используется в числовых индикаторах и итоговых сводках.

```text
COUNTD([username])
```

## Количество активных дней

Используется для оценки числа календарных дней, в которые наблюдалась активность.

```text
COUNTD([date_only])
```

## Количество сессий

Используется в числовых индикаторах и сводках на основе таблицы `listening_sessions.csv`.

```text
COUNTD([session_id])
```

## Доля прослушиваний

Используется для расчёта доли категории в общем объёме прослушиваний.

```text
SUM([plays_count]) / TOTAL(SUM([plays_count]))
```

## Порядок дней недели

Для корректной сортировки дней недели используется числовое поле `weekday_number`.

Соответствие значений:

```text
0 = Понедельник
1 = Вторник
2 = Среда
3 = Четверг
4 = Пятница
5 = Суббота
6 = Воскресенье
```
